dboydgit | Free Code Camp | Image Search

How to use:

Pass a search string to "https://db-fcc-img.glitch.me/img/<mySearchString>", the service will return a JSON string object containing the image url, description, and the site url where the image is hosted for up to 10 results.

Offset the search results by adding "?offset=2" to the url (or any offset number)

Example:

input: https://db-fcc-img.glitch.me/img/cats
